## 🌟 核心功能

### 📈 **智能投资建议**
```
当前价格 < 历史均值 × 95% → 🟢 建议买入
当前价格 > 历史均值 × 105% → 🔴 建议卖出  
其他情况                    → 🟡 建议持有
```

### 💡 **实时基金监控**
- ✅ 获取最新基金净值和涨跌幅
- ✅ 计算多种时间周期的历史均值
- ✅ 自动标记异常波动和投资机会
- ✅ 支持批量监控，性能优异

### 📋 **个性化配置**
- 🎛️ 灵活的基金代码管理
- 🎯 持有基金与关注基金分组
- ⚙️ 可调节的买卖阈值参数
- 📝 详细的决策置信度评分

## 🚀 快速开始

### 环境要求
- Python 3.8+ 
- 网络连接（用于获取基金数据）

### 一键启动
```bash
# 1. 克隆项目
git clone https://github.com/yourusername/sora.git
cd sora

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置基金列表
vim config/funds.yaml

# 4. 启动分析
python main.py
```
